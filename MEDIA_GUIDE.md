## Image Specifications

### Recommended Sizes:
- **Hero Background**: 1920x1080px or higher
- **Gallery Images**: 1200x1200px (square format works best)
- **Merch Photos**: 800x800px (square format)
- **Video Thumbnails**: 1280x720px (16:9 aspect ratio)
- **Logo**: 400x100px (PNG with transparent background)

### File Formats:
- **Images**: JPG, PNG, WebP
- **Videos**: MP4, WebM
- **Audio**: MP3, WAV
- **Icons**: ICO, PNG, SVG

## Current Fallbacks

Until you add your own media, the website will show:
- Placeholder images for missing content
- Your logo will show "OTAWANTA" text
- Default gradients and patterns for backgrounds

## Tips

1. **Optimize your images** before uploading (use tools like TinyPNG)
2. **Keep file sizes reasonable** for web performance
3. **Use consistent aspect ratios** for gallery images
4. **Compress videos** to reduce loading times
5. **Test on mobile** to ensure images look good on smaller screens

Replace files as needed - the website will automatically use your new content!
